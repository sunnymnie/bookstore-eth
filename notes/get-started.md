# Getting started
1. build folder with NextJs components: `npx create-next-app bookstore-eth`
2. remove
    - `pages>api` folder
    - `styles>Home.module.css`
3. Replace `pages>index.js` with

```
export default function Home() {
  return (
    <>Hello World</>
  )
}
```

## Basic commands
- `npm run dev` to run server
